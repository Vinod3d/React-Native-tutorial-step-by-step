# Navigation

## Navigation in React Native

Navigation is essential for multi-screen applications. React Native Navigation provides several navigation patterns:

- **Stack Navigation**: Standard navigation pattern with back button
- **Tab Navigation**: Bottom or top tab-based navigation
- **Drawer Navigation**: Slide-in drawer menu

## React Navigation

React Navigation is the most popular navigation library for React Native.

### Installation

```bash
npm install @react-navigation/native @react-navigation/stack
```
