# API Integration

## Making HTTP Requests

React Native provides two main methods for API calls:

1. **Fetch API**: Built-in browser standard
2. **Axios**: Popular third-party library

## Key Concepts

- Async/await syntax
- Error handling
- Loading states
- Data fetching in useEffect

See [fetch-example.js](fetch-example.js) and [axios-example.js](axios-example.js) for implementations.
