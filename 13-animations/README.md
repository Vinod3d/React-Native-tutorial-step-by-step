# Animations

## Adding Motion to Your App

Animations enhance user experience. React Native provides:

- **Animated API**: Low-level animation control
- **LayoutAnimation**: Automatic layout transitions
- **React Native Reanimated**: Powerful animation library

## Common Use Cases

- Fade in/out
- Slide animations
- Scale transforms
- Rotate transforms

Check [animated-api/](animated-api/) for examples using the Animated API.
