# Styling & Flexbox

## Introduction to Styling

React Native uses a subset of CSS properties for styling components through StyleSheet.

## Flexbox Layout

Flexbox is the primary layout system in React Native:

- **flex**: Defines how components grow or shrink
- **flexDirection**: Sets main axis direction (row/column)
- **justifyContent**: Aligns items along main axis
- **alignItems**: Aligns items along cross axis
- **alignSelf**: Individual item alignment
